# 댓글 수정/삭제 + 관리자 댓글 관리 + 아이콘 적용

## 요약
- 공개 블로그 상세에서 **본인 댓글 수정/삭제** 기능을 추가했다.
- 관리자 블로그 상세 드로어에서 **게시글별 댓글 조회/삭제**를 할 수 있도록 확장했다.
- 댓글 액션 버튼을 아이콘 기반으로 정리했다.  
  삭제는 기존 `TrashIcon`, 수정은 요청한 SVG 경로를 반영한 `NoteIcon` 사용.

## 구현 내용
- 공개 댓글 API 확장
  - `PATCH /api/comments/[id]`: 본인 댓글 수정
  - `DELETE /api/comments/[id]`: 본인 댓글 삭제
  - 인증 사용자와 댓글 작성자 일치 검증 추가
- 댓글 저장소 함수 확장 (`lib/comments/repository.ts`)
  - `updateCommentByAuthor`
  - `deleteCommentByAuthor`
  - `getCommentsByPostIdForAdmin`
  - `deleteCommentByIdForAdmin`
  - 댓글 타입에 `updatedAt` 매핑 추가
- 공개 댓글 UI 확장 (`components/blog/comments-section.tsx`)
  - 본인 댓글에 수정/삭제 액션 노출
  - 인라인 편집(저장/취소) 흐름 추가
  - 수정된 댓글 `수정됨` 배지 텍스트 노출
  - 성공/오류 메시지 흐름과 댓글별 pending 상태 분리
- 관리자 댓글 관리 API 추가
  - `GET /api/admin/posts/[id]/comments`
  - `DELETE /api/admin/comments/[commentId]`
- 관리자 블로그 드로어에 댓글 관리 섹션 추가 (`components/admin/blog/blog-manager.tsx`)
  - 선택 게시글의 댓글 목록 로드
  - 댓글 즉시 삭제 지원
- 아이콘 반영
  - `components/ui/icons/note-icon.tsx`: 요청한 수정 SVG로 교체
  - `components/blog/comments-section.tsx`: 수정 버튼 `NoteIcon`, 삭제 버튼 `TrashIcon` 적용
