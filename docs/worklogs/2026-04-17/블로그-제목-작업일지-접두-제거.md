# 블로그 제목 작업일지 접두 제거

## 작업 배경
- 블로그 제목에 `작업일지 - 2026-XX-XX` 같은 접두가 노출되어 가독성이 떨어지는 문제를 수정.

## 변경 내용
- 파일: `scripts/generate-docs-blog-reseed-sql.mjs`
- 제목 정규화 로직 추가:
  - `작업일지 - YYYY-MM-DD (주제)` -> `주제`
  - `작업일지 - YYYY-MM-DD 주제` -> `주제`
  - `작업일지 (YYYY-MM-DD) 주제` -> `주제`
  - `작업일지 - 주제` -> `주제`
- 헤더 제목이 비정상일 때 파일명 기반 fallback도 접두 제거 후 생성하도록 보강.

## 재생성
- `node scripts/generate-docs-blog-reseed-sql.mjs --bootstrap-translations`
- `node scripts/generate-docs-blog-reseed-sql.mjs`

## 확인 결과
- SQL 내 KO 제목 샘플에서 날짜 접두 제거 확인
  - `테마 스무딩 + KO/EN/JA i18n`
  - `오버레이 + 애니메이션 + 로케일 JSON`
  - `Admin CMS + Supabase v1`
- EN/JA 번역 제목도 동일하게 접두가 제거된 상태로 재생성됨.
