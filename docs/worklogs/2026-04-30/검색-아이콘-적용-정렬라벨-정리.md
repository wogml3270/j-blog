# 검색 아이콘 적용 + 정렬 라벨 정리

## 변경 목적
- 검색 버튼 시인성을 높이기 위해 공통 검색 아이콘을 추가하고, 정렬 select 앞의 불필요한 `정렬` 텍스트 라벨을 제거한다.

## 적용 내용
- `components/ui/icons/search-icon.tsx` 신규 추가
  - 제공한 SVG 기반 `SearchIcon` 컴포넌트 생성

- 검색 버튼 아이콘 적용
  - `components/ui/content-search-toolbar.tsx`
    - submit 버튼에 `SearchIcon` 추가
  - `components/admin/blog/blog-manager.tsx`
    - 관리자 블로그 검색 버튼에 `SearchIcon` 추가
  - `components/admin/projects/projects-manager.tsx`
    - 관리자 프로젝트 검색 버튼에 `SearchIcon` 추가

- 정렬 라벨 텍스트 제거
  - `components/ui/content-search-toolbar.tsx`
    - `sortLabel` prop 제거
    - sort select은 텍스트 span 없이 렌더
  - `app/[lang]/(site)/blog/page.tsx`
  - `app/[lang]/(site)/projects/page.tsx`
    - `sortLabel` 전달 제거

- i18n 키 정리
  - `locales/ko/blog.json`, `locales/en/blog.json`, `locales/ja/blog.json`
  - `locales/ko/projects.json`, `locales/en/projects.json`, `locales/ja/projects.json`
  - `sortLabel` 키 삭제

## 검증
- `npx tsc --noEmit` 통과
- `npm run lint` 통과
