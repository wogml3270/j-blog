# 관리자 상세 드로어 가로 오버플로우 수정

## 배경
- 관리자 블로그/프로젝트 상세 수정 드로어에서 일부 입력 행이 가로 폭을 초과하며 오버플로우가 발생했다.
- 특히 `datetime-local + 해제 버튼`, `라벨 + 탭/체크박스` 조합에서 `w-full` 입력과 `nowrap` 정렬이 겹치며 폭 충돌이 났다.

## 수정 내용
- `components/admin/common/editor-drawer.tsx`
  - 드로어 본문 래퍼에 `min-w-0`, `overflow-x-hidden` 추가.
- `components/admin/blog/blog-manager.tsx`
  - 폼/fieldset/카드에 `min-w-0` 추가.
  - `콘텐츠 언어`, `슬러그` 헤더 행을 `flex-wrap` 기반으로 변경.
  - 예약 발행 입력 영역을 `flex-wrap` + `input flex-1` 구조로 변경.
  - 읽기 전용 게시일 입력은 `max-w-sm`로 제한.
- `components/admin/projects/projects-manager.tsx`
  - 폼/fieldset/카드에 `min-w-0` 추가.
  - `콘텐츠 언어`, `슬러그` 헤더 행을 `flex-wrap` 기반으로 변경.
  - 관련 링크 입력 그리드의 4열 전환 breakpoint를 `sm` -> `md`로 조정해 협소 폭에서 줄바꿈되도록 수정.

## 검증
- `npx tsc --noEmit` 통과
- `npm run -s lint` 통과
