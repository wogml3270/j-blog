# 댓글 공통 컴포넌트 블로그/프로젝트 통일

## 작업 배경
- 프로젝트 상세 댓글 UI가 블로그와 다르게 렌더되어 유지보수 비용이 증가했습니다.
- 동일한 UX를 유지하기 위해 댓글 섹션을 공통 컴포넌트로 통합했습니다.

## 주요 변경 사항
- `components/comments/shared-comments-section.tsx` 신규 생성
  - 로그인 상태 조회, 댓글 작성/수정/삭제, 성공 메시지 자동 제거(3초), 스켈레톤 UI를 공통 처리
  - API 엔드포인트와 slug 키(`postSlug`/`projectSlug`)를 props로 받아 재사용 가능하게 구성
- `components/blog/comments-section.tsx`
  - 블로그 전용 구현 제거
  - 공통 컴포넌트 래퍼로 교체
- `components/project/comments-section.tsx`
  - 프로젝트 전용 구현 제거
  - 블로그와 동일 UI/동작을 사용하는 공통 컴포넌트 래퍼로 교체

## 함께 정리한 항목
- `components/admin/projects/projects-manager.tsx`
  - 제거된 도메인 필드(성과/주요 기여) 잔여 코드 제거
  - 관련 링크 `isPublic` 입력/수정 UI 보강
  - 프로젝트 상세 드로어 내부 댓글 관리 섹션 추가
- `lib/projects/data.ts`
  - 링크 타입(`isPublic`) 반영
  - 제거된 프로젝트 필드 반환 제거
